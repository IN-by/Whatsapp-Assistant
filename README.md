# WhatsApp Assistant 插件

## 安装指南

**⚠️ 注意：** 由于我们是内部使用插件（未发布到 Chrome 应用商店），您的同事需要开启“开发者模式”才能安装。这是 Chrome 的安全限制。

### 第一步：下载
1. 访问我们项目的 [Releases 页面](https://github.com/IN-by/Whatsapp-Assistant/releases)。
2. 下载最新版本（例如 v1.0）附件中的 `extension.crx` 文件。

### 第二步：开启开发者模式
1. 在 Chrome 浏览器地址栏输入 `chrome://extensions` 并回车。
2. 在页面右上角，找到 **"开发者模式" (Developer mode)** 开关，点击**打开**。

### 第三步：拖拽安装
1. 打开您下载了 `extension.crx` 的文件夹。
2. 将 `extension.crx` 文件直接**拖入**刚刚打开的 Chrome 扩展管理页面中。
3. Chrome 会弹出提示 "要添加...吗？"，点击 **"添加扩展程序"**。

## 常见问题

**Q: 为什么要开开发者模式？**
A: 因为我们没有向 Google 缴纳 $5 注册费并经过他们的商店审核流程。对于私有/内部工具，这是标准做法。

**Q: 安装后还需要手动更新吗？**
A: 不需要。只要您保留插件不被卸载，浏览器会根据我们在后台配置的自动更新机制，定期检查 GitHub 上的新版本并自动升级。
